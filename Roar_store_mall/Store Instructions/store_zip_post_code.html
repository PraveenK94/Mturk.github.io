INSTRUCTIONS FOR FETCHING ZIPCODE

Postal Code - Zip code will be in a (5-4) digit format. Where the first 5 digits will be available in the store website and the rest 4 digit to be fetched using the USPS Validator link mentioned above and follow the below sub step:

 

USPS Validator link - Click here

 

While fetching zipcode through USPS validator make sure to use the store name in "company blank" (For Ex: “ Claire’s” in “company name” , address line 1 in the "Street Address" (for ex: “6800 OXON HILL ROAD”) and suite numbers to be used in "suite/apt/other" (For ex: “SPC 855”, make sure to use the suite numbers to fetch the zipcode, if the suite number is provided in the website) followed by city,state and 5digit zipcode”.

Note- Any miss on inputting store name in “Company” , Address line 1 and 2 for “Street address” &” suite/apt/other” respectively would result in incorrect postal code. The address result from USPS validator should match with the address you input for the stores.

In the company name blank (use the store name as provided in the Input field including special characters), followed by address line 1 and line 2, state, city and first 5 digit zipcode (taken from the store website). Refer the below attached image.
