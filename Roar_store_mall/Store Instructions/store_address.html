INSTRUCTIONS FOR FETCHING STORE ADDRESS

1) Go to official store website provided as part of inflow and capture the store's address for the respective Mall, state and city. (Important: Address information should be collected from official store website and not from the Google search results.)

2) Go to retailer’s official website and look for “Store Locator” or “Store Directory”. Please ensure the location is United States.

Note- The Example and navigation provided below(in screenshots) are using a website https://stores.claires.com/us/ . Depending on the stores you receive the navigation through the website would vary.

Example- As highlighted below in yellow > Please select United Sates as location and Find a Store option. The option can vary depending on the website. Please go through the different website page to locate the store for the mall and zip code provided in the inflow.



 

3) Enter the City, State OR Zip code of the mall provided in the inflow . You can browse through state option and select the right city and Mall.


 

4) Select the correct store location and find the store address (You may have to look through multiple pages on the store’s website to find the address), make sure the mall name, city, state and zipcode in the result matches with the provided inflow.

For the below example let’s say the inflow was State > Alaska , City > ANCHORAGE , Mall > 5TH AVENUE MALL.

• Once you find the right Mall click on More details option



 

Filling Address line 1 & 2: (Incorrect capturing of Address line 1 and 2 would result in lower quality)

a) Capture street number & name in Address line 1 field. In the above example for 5th Avenue Mall- 320 W 5TH Avenue would be captured in Address line 1

b) Capture the suite number or store number or building number in ‘address line 2’.In the above example for 5th Avenue Mall “#148” would be in Address line 2. If no suite number is provided, then ‘address line 2’ can be left blank.

Note: There is no need of capturing City/ State / Zipcode in Address line 1 & 2.

While collecting the Address lines from official store website " Capturing of city name , state name and zipcode is not required", collect only the street names in address line 1 and suite numbers in "address line 2". For ex ("6800 OXON HILL ROAD SPC 855" , consider "6800 OXON HILL ROAD - address line 1" & "SPC 855 - address line 2" and address line 3 can be left blank. In few cases the address line 2 will be like this "SPC 800"/ "STE E"/"BLDG 10"/"Unit 20", also do not consider the range number which is provided before the street name as "suite number".
